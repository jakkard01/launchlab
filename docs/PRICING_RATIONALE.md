# Pricing Rationale

## Principios
- Precios en EUR con rangos orientativos basados en esfuerzo real (horas/mes), revisiones y complejidad.
- “Desde…” siempre con caps claros (duracion, volumen, formatos, integraciones).
- Subidas de precio cuando hay regrabaciones, integraciones complejas o SLA mas estricto.

## Web
- Web Starter: 1 pagina (6-8 secciones), 1 ronda, SEO base + performance.
- Web Growth: 4-6 paginas, 2 rondas, integraciones basicas + analitica.
- Web Pro: 6-10 paginas, 3 rondas, variantes de hero/CTA.
- Add-on Ecommerce: catalogo + carrito/checkout o pedido por WhatsApp, panel basico.
- Rationale: mas paginas + revisiones + integraciones aumentan horas de diseno, copy y QA.

## Video Packs
- Limite: hasta 45s por video, 2 revisiones por pieza.
- Growth: 8 videos/mes, 2 formatos, 1 concepto base + variaciones.
- Pro: 12-16 videos/mes, 3 formatos, 2 conceptos base, storyboard simple.
- Rationale: mas volumen + formatos + guion y coordinación => mas horas de edición + PM.

## Doblaje + Subtitulos
- Incluye voiceover + subtitulos pro (2 idiomas).
- Lip-sync basico solo talking head; avanzado es add-on.
- Rationale: costo por minuto de audio + QA + subtitulado + coordinacion por lote.

## n8n Ops
- Simple: 1-2 integraciones, <10 nodos.
- Compleja: 3+ integraciones, 20+ nodos, ERP/CRM custom + logs/retries.
- Rationale: discovery + diseño + QA + monitoreo aumenta horas cuando hay branching y errores.

## Bots
- Canales soportados: web widget + WhatsApp Cloud API.
- Alcance por intents/flows + handoff.
- Rationale: diseño conversacional + integraciones + QA + handoff.

## Super Pack
- Precio orientativo por caps fijos:
  - 8 videos/mes (≤45s, 2 formatos)
  - 2 flujos n8n/mes
  - 1 bot (≤15 intents + handoff)
  - 2 doblajes/mes (2 idiomas)
  - SLA 24–48h habiles
- Rationale: bundle con volumen controlado que permite cumplir sin sobrecarga.
